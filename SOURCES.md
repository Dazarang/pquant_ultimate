# Sources

## Autoresearch

- [karpathy/autoresearch](https://github.com/karpathy/autoresearch) -- Original concept. Agent-driven loop editing train.py, single metric ratchet (val_bpb), results.tsv tracking.
- [trevin-creator/autoresearch-mlx](https://github.com/trevin-creator/autoresearch-mlx) -- MLX port for Apple Silicon. Smaller defaults, AdamW-only.
- [buildoak/tennis-xgboost-autoresearch](https://github.com/buildoak/tennis-xgboost-autoresearch) -- XGBoost + tabular data adaptation. Bash outer loop, directive rotation, combat log, anti-gaming gates, immutable eval separation. Closest to our design.
